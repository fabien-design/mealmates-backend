# MealMates

### Initialisation du projet 
```bash
    make install
```

### JWT generer ses clés:
```bash
    make generate-keys
```
utilisez un .env.local pour stocker vos données privées.

### Definir l'url du front (react)
Modifier le .env.local 
```bash
    FRONTEND_URL=https://your-site.com
```
